## ✅ Step 7: Add Products & Collections

1. **Add Products**
   - [ ] Go to **Products → Add product**
   - [ ] Or import via CSV

2. **Create Collections**
   - [ ] Go to **Products → Collections**
   - [ ] Create: Best Sellers, New Arrivals, etc.

3. **Update Featured Sections**
   - [ ] In Customizer, go to Homepage
   - [ ] Select actual collections for featured sections

---

## ✅ Step 8: Final Pre-Launch Checks

- [ ] All navigation links work
- [ ] Contact form submits successfully
- [ ] Mobile layout looks correct
- [ ] All images load properly
- [ ] Page load time acceptable (<3 seconds)

---

## ✅ Step 9: Go Live

1. [ ] **Publish Theme:** Online Store → Themes → Publish
2. [ ] **Remove Password:** Online Store → Preferences → Uncheck password
3. [ ] **Configure Payments:** Settings → Payments
4. [ ] **Configure Shipping:** Settings → Shipping
5. [ ] **Test Order:** Place and refund a test order
